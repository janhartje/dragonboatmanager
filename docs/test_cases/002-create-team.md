# Test Case: Create New Team

**ID**: TC-002
**Description**: Verify that a user can create a new team and is onboarded correctly.
**Pre-conditions**:
- User is logged in.
- User has no teams or chooses to create a new one.

**Steps**:
1. Click "Team erstellen" on the dashboard.
2. Enter Team Name "Exploratory Test Team".
3. Click "Erstellen".
4. (If applicable) Complete "Welcome to Team" profile setup (Skill selection).
5. Verify redirection to Team Dashboard.
6. Verify Team Name header.

**Expected Result**:
- Team is created.
- User is made Captain (implied).
- Team Dashboard is visible with correct name.
