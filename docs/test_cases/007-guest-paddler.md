# Test Case: Add Guest Paddler

**ID**: TC-007
**Description**: Verify that a guest paddler can be added and assigned to a boat.
**Pre-conditions**:
- User is in Planner mode for an event.

**Steps**:
1. Click "Gast+" button.
2. Enter Name "Guest Paddler" (or similar).
3. Enter Weight "90".
4. Click "Hinzufügen".
5. Drag/Click guest to a seat.

**Expected Result**:
- Guest appears in "Verfügbar" list.
- Guest can be assigned to a seat.
