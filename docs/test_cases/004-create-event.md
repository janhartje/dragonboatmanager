# Test Case: Create Training Event

**ID**: TC-004
**Description**: Verify that a user can create a new training event.
**Pre-conditions**:
- User is logged in as Team Owner/Admin.
- Team Dashboard is open.

**Steps**:
1. Click "Neuer Termin".
2. Enter Title "Test Training".
3. Select Date and Time.
4. Select Type "Training".
5. Click "Termin erstellen".
6. Verify event appears in the event list/dashboard.

**Expected Result**:
- Event created successfully.
- Event listed with correct details.
