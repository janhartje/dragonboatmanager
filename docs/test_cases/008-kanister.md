# Test Case: Kanister (Ballast) Usage

**ID**: TC-008
**Description**: Verify that a 'Kanister' (25kg ballast) can be placed in the boat.
**Pre-conditions**:
- User is in Planner mode.

**Steps**:
1. Select "Kanister" tool/icon (or drag from sidebar).
2. Place on an empty seat.

**Expected Result**:
- Seat displays Kanister icon.
- Weight (25kg) is added to the boat calculation.
