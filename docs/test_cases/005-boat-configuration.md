# Test Case: Boat Configuration

**ID**: TC-005
**Description**: Verify that a user can assign paddlers to seats in a boat for an event.
**Pre-conditions**:
- User is logged in.
- "Test Training" event exists.
- Paddlers are available in the roster.

**Steps**:
1. Navigate to Dashboard.
2. Click "Planen >" on the "Test Training" event card.
3. Select a paddler from the roster list.
4. Click on a seat (e.g., Row 1, Right).
5. Select another paddler.
6. Click on a seat (e.g., Row 1, Left).
7. Verify both paddlers are assigned to their respective seats.

**Expected Result**:
- Paddlers are correctly assigned to the selected seats.
- Visual confirmation of the lineup is visible.
