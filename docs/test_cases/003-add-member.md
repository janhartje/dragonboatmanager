# Test Case: Add Team Member

**ID**: TC-003
**Description**: Verify that a user can add a new paddler to the team.
**Pre-conditions**:
- User is logged in and is a Team Captain or Admin.
- Team Dashboard is open.

**Steps**:
1. Click "Paddler hinzufügen".
2. Enter Name "Test Paddler".
3. Enter Weight "80".
4. Select Side Preference "Links" & "Rechts".
5. Click "Hinzufügen".
6. Verify new member appears in "Kader" list with correct details.

**Expected Result**:
- Modal opens.
- Form fields are available (Name, Weight, Side).
- Member is added to list upon submission.
